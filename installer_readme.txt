Stock Oracle — Stock Prediction & Monitoring System
=====================================================

What is this?
Stock Oracle monitors stocks in real-time, combines 38 data signals
(technical analysis, news sentiment, SEC filings, social media, and more),
and uses machine learning to predict price direction.

Features:
  - Real-time monitoring with live prices and predictions
  - Breakout Scanner — finds stocks about to make big moves
  - Machine learning that improves over time
  - Claude AI advisor for deeper analysis (optional)
  - Signal intelligence that learns which indicators actually work
  - Tracks its own accuracy and adjusts automatically

Getting started (after install):
  1. Launch Stock Oracle from your desktop or Start Menu
  2. A setup wizard will guide you through configuration
  3. Get a FREE API key at finnhub.io for real-time prices
  4. Click "Start Monitoring" — the system does the rest
  5. Click "Help" anytime for a full guide

Optional API keys (all free):
  - Finnhub (finnhub.io) — real-time stock prices
  - Anthropic (console.anthropic.com) — AI advisor (~$5/month)
  - FRED (fred.stlouisfed.org) — economic indicators
  - Reddit (reddit.com/prefs/apps) — social sentiment
  - SEC EDGAR — just needs your email

Your data (predictions, settings, ML models) is stored separately
from the app, so updates never lose your data.

DISCLAIMER: This is a research and educational tool. It is NOT
financial advice. No algorithm can predict the stock market with
certainty. Always do your own research before investing.
